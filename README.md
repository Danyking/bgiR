bgiR
====

这个R包是为华大科技服务有限公司（ http://www.bgitechsolutions.com ）为合作伙伴提供的针对生物医学研究的R语言培训服务而开发的，包括以下内容：

* R语言基础
* R语言绘图基础
* 数据挖掘入门
* 单元测试

各项培训的演示文件、例子和所使用的数据都在这个包中。
